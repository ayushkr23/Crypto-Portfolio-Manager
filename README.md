# 🚀 Crypto Portfolio Manager

A modern, interactive, and simple **Cryptocurrency Portfolio Management** web app to track real-time crypto prices, visualize 24H trends, and get assistance through an AI-powered chatbot.

---

## 📖 About the Project

**Crypto Portfolio Manager** is a responsive and lightweight web app designed to:

* 📈 Display real-time cryptocurrency prices using  **CoinGecko API** .
* 📊 Show mini price charts for 24H performance.
* 🤖 Provide a **Gemini AI Chatbot Assistant** to answer cryptocurrency-related queries.
* 💬 Allow users to get quick insights like live price changes, portfolio tracking tips, and more.

Built with a clean, minimal, and futuristic UI using **pure HTML, CSS, and JavaScript** — no heavy frameworks!

---

## 🎯 Features

* 🔥 **Live Cryptocurrency Prices** (Top 10 by market cap)
* 📉 **24H Price Change Line Charts**
* 🤖 **Gemini API Powered Chatbot**
* 🌗 **Dark-themed, responsive design**
* 📱 **Mobile-friendly UI**
* 📜 **About Page** with team info and disclaimer
* 📬 **Contact Section** for queries

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3** (Responsive + Animated UI)
* **JavaScript (ES6)**
* **CoinGecko Public API** (For real-time prices)
* **Google Gemini API** (For AI Chatbot)
* **Chart.js** (For live sparkline charts)
* **FontAwesome** (For icons)
